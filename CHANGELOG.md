## [0.1.0] - 2025-12-04

### Added
- Global hotkey registration and execution on macOS
- TOML-based config file support (`~/.config/hopkey.toml`)
- Launch agent installation for automatic startup (`hopkey install`)
- Launch agent uninstallation (`hopkey uninstall`)
- Launch agent restart for config reload (`hopkey restart`)
- Custom config file path support via `--config` flag
